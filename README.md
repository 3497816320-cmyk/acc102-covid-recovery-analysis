# COVID-19 Economic Recovery Analysis

## 1. Problem
This project analyses which countries recovered the fastest economically after COVID-19 using GDP data.

## 2. Data
- Source: World Bank
- Indicator: GDP (current US$)
- Period: 2019–2023

## 3. Methods
- Data loading and structuring using pandas
- Calculation of GDP growth rate
- Ranking countries by recovery speed
- Visualisation using bar charts and line charts

## 4. Key Findings
- All countries show positive recovery after COVID-19
- Emerging economies such as India and China have higher growth rates
- Developed countries show stable but slower growth

## 5. Project Structure
- covid_recovery_analysis.ipynb: main analysis
- README.md: project description

## 6. How to Run
Open the notebook and run all cells in order.

## 7. Conclusion
This project provides insights into global economic recovery trends and highlights differences between developed and emerging economies.

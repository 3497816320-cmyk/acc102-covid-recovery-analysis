# COVID-19 Economic Recovery Analysis

## 1. Problem & User

This project analyses which countries recovered the fastest economically after the COVID-19 pandemic using GDP data. The target users are students and beginners who want to understand economic recovery trends through simple and interpretable data analysis.

## 2. Data

* Source: World Bank (GDP, current US$)
* Period: 2019–2023
* Accessed: April 2026

The dataset includes selected countries and compares GDP before and after the pandemic to evaluate recovery performance.

## 3. Methods

The analysis was conducted using Python. First, the dataset was constructed and organised using pandas. Then, GDP growth rates were calculated to measure recovery speed across countries. The data was also sorted and ranked to identify which countries recovered faster. Finally, matplotlib was used to create visualisations, including a bar chart and a line chart, to present the results clearly.

## 4. Key Findings

* All selected countries show economic recovery after COVID-19
* Emerging economies (e.g. India and China) demonstrate higher growth rates
* Developed economies (e.g. USA and Germany) show more stable but slower growth
* The results suggest that economic structure may influence recovery speed
